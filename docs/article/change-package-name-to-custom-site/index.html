パッケージ名を変更しました

思いっきりました。

もともとrocuという名前付けていたんですけど、仮称だったんですよね。

いくつかのパターンを考えていて、

  • 記録: kiroku, record
  • 録: roku
  • 印字: press

とかとかそんな感じでやって、大体名前は取られているので、じゃあ、classをklassってもじる感じで rokuをrocuにすっかー、って思ってたんですけど、全然ぴーーーーんとこなくてモヤッとしていたんですよね。

で、いろいろ考えてカスタムテンプレートとかカスタムコンポーネントとか作ってたら、 カスタムサイトならピンとくるくね、と思ってとりあえず名前を調べたら誰もとってなかったんですよね。

（多分海外のカスタムの意味と日本人の考えるカスタムの意味が違う。「慣習」とかだし）

CLIもあるので、キャメルケースだと辛いので、単連結方式の名前で取得。 ついでにドメインもあるかなーと思ったら、取られていなかったので、custom-site.orgを買いました。

名前は先着なので、GithubのOrganization(https://github.com/custom-site)も取っておいた（大盤振る舞いだな）。

ワイ一人だけど楽しいから気長にやるマン。
